git

1.git checkout -b dev origin/dev————–>本地同步创建码云分支

2.git branch master ——–>本地创建自己的分支

3.git合并

3.1.在自己的分支上做提交

git add ./ -------> 提交所有的代码
git commit -m '注释'

3.2.切换到dev分支

git checkout dev

3.3.获取最新代码

git pull origin dev

3.4.合并自己的分支itcast1到dev分支

git merge itcast1

3.5.提交到服务器gitee.com

git push origin dev

4.直到所有人都合并完成

4.1.获取最新代码

git pull origin dev

4.2.切换到自己的分支

git checkout itcast1

4.3.合并dev分到到自己的分支

git merge dev

5.git历史记录

git reflog-------------->查看历史版本
git reset 版本号-------->回到指定的版本号
git status-------------->查看文件变化
git checkout -- 文件名-->恢复指定的文件
